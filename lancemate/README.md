# LANCEmate

This tool exists to simplify debugging, tuning, and configuration of UniLANCE. Currently is 
just a default Tauri app template, for later.

The app is supposed to be as cross-platform as possible, aka it's able to run both on my
development machine, and on my phone, so I can do all these curious tasks on the go. The
project itself though is split into two components, the CLI portion and app portion.
CLI is mainly used for automatic tasks, like generating firmware update packages, and
base filesystem images.
