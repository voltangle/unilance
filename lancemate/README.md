# LANCEmate

This tool exists to simplify debugging, tuning, and configuration of UniLANCE, made with
Kotlin Multiplatform and Compose.

The app is supposed to be as cross-platform as possible, aka it's able to run both on my
development machine, and on my phone, so I can do all these curious tasks on the go. There
will be a separate "half" of this project, aptly named "lancemate-cli", which will do all the
automation tasks, like generating firmware update packages, filesystem images, etc etc.

As of now, iOS build is broken, so for now it will just work on desktop and Android.
